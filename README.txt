-Unzip the program files:
    1. From the command line: $ unzip hw1.zip

-In order to run the two proof of function programs, insertionsort.py and mergesort.py:
    1. Make sure the two program files are located in a folder with the data.txt file containing integers to be sorted.
    2. To run insertion sort: $ ./insertionsort.py
    3. To run merge sort: $ ./mergesort.py
    4. Result files named insertionsort.txt and mergesort.txt will be created in the current folder.

-In order to run the two timing programs, insertTime.py and mergeTime.py:
    1. To run insertion sort timing: $ ./insertTime.py
    2. To run merge sort timing: $ ./mergeTime.py
    3. Timing results will be displayed as terminal output.
